# AI-Powered Telugu Marketing Content Generator

A web application that generates marketing content in Telugu using AI. This project features a React frontend and a Python backend.

## 🛠 Tech Stack
* **Frontend:** React (Yarn)
* **Backend:** Python (FastAPI)

## 📋 Prerequisites
Make sure you have the following installed on your local machine:
* Node.js & Yarn
* Python 3.8+

---

## 🚀 Installation & Setup

### 1. Clone the repository
\`\`\`bash
git clone https://github.com/Prashanthanuga/AI-Powered-Telugu-Marketing-Content-Generator-Web-Application-.git
cd AI-Powered-Telugu-Marketing-Content-Generator-Web-Application-
\`\`\`

### 2. Backend Setup
Open a terminal and navigate to the backend folder:
\`\`\`bash
cd backend

# Install the required Python packages
pip install -r requirements.txt

# Start the backend server
python server.py
\`\`\`
*(Note: You will need to create a `.env` file in the `backend` folder and add your specific API keys before running the server.)*

### 3. Frontend Setup
Open a second terminal and navigate to the frontend folder:
\`\`\`bash
cd frontend

# Install the required Node packages
yarn install

# Start the frontend development server
yarn start
\`\`\`
*(Note: You will also need to create a `.env` file in the `frontend` folder if your React app requires any specific environment variables.)*
